# AQI Analysis - Delhi

## Overview
This project analyzes Delhi's Air Quality Index (AQI) using basic data science techniques. It includes data visualization and a simple machine learning model to observe trends and predict future AQI values.

## Features
- Data preprocessing using Pandas  
- Visualization of AQI trends  
- Simple AQI prediction using Linear Regression  
- Easy-to-understand and beginner-friendly implementation  

## Tech Stack
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  

## Project Structure
aqi-analysis-delhi/

## How to Run
1. Clone the repository:
   git clone https://github.com/aky35653-sketch/aqi-analysis-delhi

2. Install dependencies:
   pip install pandas matplotlib seaborn scikit-learn

3. Run the project:
   python main.py

## Output
- Graph showing AQI trends  
- Predicted AQI value using a simple ML model
